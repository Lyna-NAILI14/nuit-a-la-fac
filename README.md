# Nuit à la Fac

 Projet de fin d'année – Licence 3 MIAGE (Université d'Aix-Marseille)

## Description

"Nuit à la Fac" est un jeu d'aventure développé en Java dans le cadre d'un projet de Licence 3 MIAGE.

Le joueur incarne un étudiant qui se retrouve enfermé dans son université après s'être endormi en cours. Pour s'échapper avant le lever du jour, il doit explorer les différentes salles, résoudre des énigmes, récupérer des objets, gérer son énergie et interagir avec différents personnages.

## Fonctionnalités du jeu

- Authentification des joueurs
- Création de comptes
- Sauvegarde et chargement des parties
- Gestion de l'inventaire
- Gestion de l'énergie
- Déplacements entre les salles
- Résolution de 5 énigmes
- Interface graphique Java Swing
- Base de données SQLite
- Tests unitaires

## Technologies utilisées

- Java
- Java Swing
- Maven
- SQLite
- JUnit
- Git / GitHub
